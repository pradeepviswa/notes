## make inxex.html accessible to all
- Amazon S3 > Buckets > my-bucket-demo-1919
- Permissions (tab)
- Block public access (bucket settings) > Edit > Remove "Block all public access"
- <img width="661" height="325" alt="image" src="https://github.com/user-attachments/assets/24dff8cf-79fc-459b-bdf6-cb9e8110e3ba" />
- Access control list (ACL) > Edit > selec "ACLs enabled" > save changes
- Objects (tab) > select index.html > Actions > Make public using ACL > Make Public
- <img width="1300" height="435" alt="image" src="https://github.com/user-attachments/assets/218cc594-9e82-4143-b4f4-f6c25429b514" />
- click on "index.html" cli copy Object URL, then browse, example: https://my-bucket-demo-1919.s3.us-east-1.amazonaws.com/index.html
