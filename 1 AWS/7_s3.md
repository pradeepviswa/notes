## Copy Content in s3 bucket from ec2
### Create IAM Role with S3 access
- IAM > Roles > Create role
- Trusted entity type: AWS service
- Use case: EC2
- Next
- Permissions policies: AmazonS3FullAccess
 - Else can write custom policy
 ```
{
  "Version": "2012-10-17",
  "Statement": [
    {
      "Effect": "Allow",
      "Action": "s3:GetObject",
      "Resource": "arn:aws:s3:::my-test-bucket-pv-0909/*"
    }
  ]
}
 ```
- Next
- Role name: ec2-fullpermission-role
- Create Role

### Attach Role to EC2 Instance
- EC2 > Instances
- select instance > Actions > Security > Modify IAM Role
- <img width="489" height="239" alt="image" src="https://github.com/user-attachments/assets/d4129120-9a7d-445f-8b0e-81d97662c0a7" />
- IAM role: select "ec2-fullpermission-role"
- Update IAM Role

### Access File from S3
- Download file to current folder
  ```
  aws s3 cp s3://my-test-bucket-pv-0909/file1.txt .
  ```
- List files in bucket
  ```
  aws s3 ls s3://my-test-bucket-pv-0909
  ```
- Upload file in bucket
  ```
  aws s3 cp file1.txt s3://my-test-bucket-pv-0909/
  ```
- Upload folder recursively
  ```
  aws s3 cp myfolder s3://my-test-bucket-pv-0909/ --recursive
  ```
- Sync EC2 → S3
  ```
 aws s3 sync myfolder s3://my-test-bucket-pv-0909/
  ```
- S3 → EC2
 ```
aws s3 sync s3://my-test-bucket-pv-0909 myfolder
 ```
